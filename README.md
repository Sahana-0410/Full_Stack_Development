# 🎓 College Course Enrollment Tracker

A simple, interactive Angular application built to help students manage their semester course planning.

## ✨ Features
* **Live Search:** Quickly filter through courses by name or department category.
* **Smart Enrollment:** Toggle enrollment status with a single click.
* **3-Course Limit:** Built-in logic prevents students from selecting more than 3 courses.
* **Top-Right Dashboard:** Real-time counter showing enrollment status (e.g., 2 / 3).

## 🚀 Technical Stack
* **Framework:** Angular 17+ (Standalone Components)
* **Logic:** Custom Pipes for filtering and Getters for state management.
* **UI:** Clean, responsive CSS with dynamic color-coding for enrolled courses.
